# PalOps Web 1.3.0 完整功能参考

> 语言：**简体中文** | [English](features.en.md)

本文按 1.3.0 当前路由与服务端权限边界描述全部公开功能。界面截图位于 [`images/zh-CN/`](images/zh-CN/)，全部来自当前 Vue 页面并使用虚构演示数据。

## 服务器态势

### 运行概览

集中展示 PalServer 进程状态、主机 CPU/内存/磁盘、在线人数、Server FPS、平台与 PalDefender 版本、存档状态和实时事件。

### 服务器统计

按小时或天汇总在线人数、FPS、CPU、内存、异常、备份和 Webhook 事件，显示数据保留范围与更新时间。

## 服务器数据

### 玩家管理

合并在线数据和只读存档索引，展示玩家等级、公会、坐标、背包、帕鲁、身份来源、最后在线时间与匹配置信度。

### 公会据点

查看公会成员、会长、据点归属证据、世界坐标、工作帕鲁和据点详情，并可直接跳转世界地图。

### 世界地图

提供 Palpagos / World Tree 离线地图、1,251 条固定 POI、94 个快速传送点、实时玩家、公会据点和自定义标记。玩家刷新支持 1/2/3/5/10/15/30 秒，默认 3 秒；据点与自定义标记独立 30 秒刷新。

## 服务器运营

### 物资发放

使用三步流程选择多个玩家、检索中文/英文/内部 ID、构建物品与帕鲁清单，并发放经验或科技点。

### 消息发送

支持全服广播、警告、定向消息、在线玩家选择、踢出和 PalDefender 配置重载等受控操作。

### RCON 控制台

提供标准 Palworld 与 PalDefender 扩展命令、能力探测、快捷模板、高风险确认、输出展示和命令历史。

## 配置与运维

### Palworld 配置中心

结构化管理 `PalWorldSettings.ini` 和启动参数，同时保留原文编辑；支持类型、范围、语法与端口冲突检查、差异预览、修改前备份、原子替换和安全重启。

### 自动任务

创建 Cron 或固定周期任务，用于自动备份、公告、保存和重启；显示风险等级、下次运行时间和执行历史。

### 维护中心与崩溃守护

把维护公告、保存、备份、停服、脚本、启动和健康验证编排为可审计流程，并提供崩溃检测、自动恢复、退避和熔断状态。

## 玩家安全

### 玩家纪律

管理白名单、封禁、在线玩家、身份关联、违规和踢出记录。白名单优先调用 PalDefender RCON 并验证文件结果，严格 UserId 校验阻止误写。

### PalDefender 防护组件

查看连接与版本，生成和维护受支持配置文件，提供字段说明、JSON/类型/路径校验、SHA-256 并发检查、备份、原子保存和 `reloadcfg` / 重启提示。

### 权限管理

管理 Owner、Administrator、Operator、Auditor、Viewer 多角色账户、启停状态和最近登录信息。

### 审计日志

记录登录、配置、进程、RCON、存档、玩家纪律、插件和权限操作的操作者、来源、结果与结构化元数据。

## 存档与数据

### 物品与帕鲁目录

使用离线目录和图标，支持分类、别名、收藏、导入、多元素搜索和数据来源说明。

### 存档备份

支持手动/自动备份、SHA-256 校验、保留策略、下载、恢复预检和受保护恢复。

### 存档差异

只读比较两次解析快照，追踪玩家、公会、据点、物品和帕鲁变化，并标记异常或高风险差异。

### 存档解析

只读解析 `Level.sav` 与 `Players` 快照，提供自动索引、格式检测、进度、失败诊断和最后成功结果回退。

## 扩展管理

### 插件与模组

管理 PalDefender、UE4SS、服务器模组和脚本的版本、哈希、依赖、兼容性、安装状态、备份和回滚信息。

## 消息中心

### 消息通知

支持企业微信、钉钉、飞书、Discord、Slack、Telegram 和通用 JSON Webhook，提供事件订阅、模板、测试和重试策略。

### 推送记录

展示投递成功、重试、失败、HTTP 状态、耗时、请求/响应摘要和错误原因，机密字段保持脱敏。

## 系统管理

### 系统日志

提供业务日志、级别筛选、全文搜索、异常详情和分页定位。

### 系统设置

集中配置 Palworld REST、PalDefender REST、RCON、存档、备份、自动任务、运行刷新和安全选项。

### 关于系统

展示 PalOps Web 版本、关键依赖、地图数据来源、许可证、项目边界和发布信息。

## 高级运维与治理

### 诊断中心

集中执行进程、REST、RCON、存档、备份、磁盘、自动任务与插件基础检查，输出分级结果、修复建议和受保护的支持包。

### 事件与故障中心

把健康异常聚合为可确认、分派、备注、恢复和重开的故障记录；规则支持严重级别、重复事件合并与审计时间线。

### 玩家洞察

基于在线会话、存档索引、纪律记录与管理员备注生成玩家时间线、活跃度和辅助风险提示。风险提示仅供人工判断，不会自动封禁。

### 世界治理

分析公会、据点和活跃度，标记长期不活跃、孤儿归属或需要人工复核的治理候选。系统不自动删除或迁移据点。

### 灾备中心

管理本地、UNC、WebDAV 与 S3 兼容目标的配置、验证和恢复演练。演练只把已校验备份复制到隔离目录，不覆盖正式存档。

### 更新中心

提供 PalServer、PalDefender 与插件组件的更新预检、风险项、审批记录和计划管理。当前仅编排预检与人工审批，不静默替换服务器程序。

### 配置版本库

为 Palworld 配置和启动参数创建不可变快照，支持哈希校验、键级差异与 Owner 精确确认回滚。

### 运维剧本

把健康刷新、备份、存档索引、通知事件和维护流程组合为白名单步骤。剧本不允许 Shell、PowerShell 或任意进程执行。

### 安全中心

管理 API 令牌策略、作用域、有效期、吊销和使用审计。令牌明文仅在创建时显示一次，持久化内容只保留 SHA-256 哈希。

### 对外集成中心

提供受限 Bearer Token API 和 HTTPS 事件订阅。订阅复用现有 Webhook 队列、重试、SSRF 校验和投递历史；外部入站事件必须使用 `external.*` 类型且不提供任意 RCON 或 Shell 执行。

## 角色与写操作

- **Owner**：全部功能，包括账户、安全配置、恢复、插件模组和高风险生命周期操作。
- **Administrator**：服务器运营、玩家、公会、配置、维护、通知与多数系统管理操作。
- **Operator**：日常运行、消息、RCON、自动任务、备份和经授权的玩家操作。
- **Auditor**：日志、审计、统计、存档差异和只读安全数据。
- **Viewer**：只读状态与基础数据。

所有实际权限以服务端授权策略为准；前端隐藏按钮不能替代后端授权。

## 数据来源

| 数据 | 主要来源 |
|---|---|
| 进程、CPU、内存、磁盘 | Windows 本机进程与系统指标 |
| 在线玩家、Server FPS | Palworld REST / PalDefender REST，按可用性降级 |
| 玩家、公会、据点、物品、帕鲁 | 本地 Palworld 存档只读解析与实时数据合并 |
| 固定地图 POI、瓦片、分类和探索状态 | Vue 前端离线静态资源 |
| 实时地图玩家、据点、自定义标记 | PalOps 后端运行时 API |
| 白名单、封禁和防护配置 | PalDefender RCON、REST 与受控 JSON 文件 |
| 设置、统计、审计和任务历史 | PalOps 本地 JSON / JSONL 数据 |

## 安全原则

写操作使用 Cookie 会话、CSRF、角色授权和审计。配置更新使用受控路径、并发哈希、修改前备份和原子替换；恢复、强停和插件安装等高风险操作要求额外权限与确认。存档差异和存档解析均为只读流程。
