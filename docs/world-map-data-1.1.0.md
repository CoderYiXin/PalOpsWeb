> 语言：**简体中文** | [English](world-map-data-1.1.0.en.md)

## 运行架构

1.1.0 将固定地图数据视为前端本地应用数据。浏览器加载图层配置、分类树、图标、搜索别名、本地化名称、探索状态和全部固定 POI；服务器 API 只提供玩家、公会据点和自定义标记。

固定数据使用 MapLibre GeoJSON source，不存在生产 PBF/fallback 切换，也不存在首次渲染时的后端地图包请求。

## 恢复默认

恢复默认只启用：

- 在线玩家；
- 离线玩家；
- 公会据点；
- 自定义标记；
- 快速传送；
- 高塔。

其他分类仍安装并可搜索，但默认关闭，避免首屏过密。

## 本地化

- `zh-CN`：中文为主显示名，英文仅作为别名和搜索词；
- `en-US`：英文主显示名；
- `ja-JP`：日文分类与显示名，稳定 ID 和坐标保持一致。

生成器拒绝多语言数量差异，前端门禁拒绝简体中文索引中的非中文主名称。

## 首次渲染优先级

玩家、公会据点、自定义标记和固定 POI 在地图组件初始化时并行请求。数据到达时如果图层骨架尚未完成，会进入待提交状态；骨架可写后立即写入，不依赖底图瓦片全部加载，也不等待定时刷新或手动切换复选框。

页面分阶段显示：地图引擎、服务器数据、服务器标记、静态数据、静态标记。

## 数据维护与验证

固定 POI JSON 直接提交到仓库，不再依赖 Python 生成器。

```powershell
cd frontend-vue
npm run verify:map-complete-local
npm run build
```

相同输入应生成相同的本地化 JSON 和清单哈希。

## World Tree 快速传送

前端索引中有 15 个快速传送点位于 World Tree 边界内。`Foot of the World Tree` 位于接近区域，坐标属于 Palpagos，因此保留在 Palpagos 图层。

## 时间显示

存档索引、公会、成员最后在线、玩家档案和概览时间统一以 `Asia/Shanghai` 显示。新生命周期时间使用明确 `+08:00` 偏移，同时保留 `DateTimeOffset` 进行绝对时间比较。
